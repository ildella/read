Welcome to FrankieGPT Web Extension. Your privacy is important to us. This Privacy Policy explains how we collect, use, and protect your information when you use our web extension.

2. Authentication

To use FrankieGPT Web Extension, users must first authenticate through https://chat.frankie-gpt.com. Authentication is required to ensure secure access and personalized service.

3. Information Collection and Use

When you press the extension icon on an active browser tab, the web extension will:

  * Collect the HTML content of the active browser tab.
  * Parse the HTML to extract relevant text information.

4. Data Security

We prioritize the security of your data. All communications between the web extension and https://chat.frankie-gpt.com are encrypted using HTTPS to protect your information from unauthorized access.
