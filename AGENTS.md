# AGENTS

Frankie Read is a new project refactored out from an old suite of tools called "Frankie Tools". 

It is a Pocket / Instappaer like app. It has a Web Browser Extension and an app, now a Progressive Web App (PWA).

The user grab articles from web pages when browsing.
The articles is parsed, cleaned and stored locally, with images.
The user then can find it in the simple app and read it in a clean format.

It did allow other small feature like "start a chat with the doc", it had a Reading Queue and a not-really-working feature where a Text To Speech model would create the audio to listen.

This new project will build on those foundations and create a standalone service with the Web Extension and the revamped App. 

See [PRODUCT.md](./PRODUCT.md) for detail about the new service vision and core principles.
